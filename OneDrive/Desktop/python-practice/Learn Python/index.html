Lesson 1: HTML - The Structure of the Web

HTML (HyperText Markup Language) is the backbone of every webpage. It defines the content and structure using various elements or "tags".

Basic HTML Structure & Essential Tags:

  • Doctype Declaration: <!DOCTYPE html>
  • HTML Structure: <html> <head> <body> ... </body> </html>
  • Semantic Elements: <header>, <nav>, <main>, <section>, <article>
  • Text Elements: <h1> to <h6>, <p>, <span>
  • Links & Images: <a href="url">, <img src="url" alt="description">
Next Lesson: CSS 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Awesome Page</title>
</head>
<body>
    <!-- Headings: H1 is most important, H6 least -->
    <h1>Welcome to My Site</h1>
    <h2>About This Page</h2>
    
    <!-- Paragraphs for text -->
    <p>This is a paragraph of text. It contains some <strong>important</strong> words and <em>emphasized</em> phrases.</p>
    
    <!-- Links: 'href' for destination, 'target="_blank"' for new tab -->
    <a href="https://www.codewithharry.com" target="_blank">Visit CodeWithHarry</a>
    
    <!-- Images: 'src' for path, 'alt' for accessibility -->
    <img src="https://placehold.co/300x200/000/FFF?text=Placeholder+Image" alt="A placeholder image" class="rounded-md my-4">
    
    <!-- Lists: Unordered (ul) and Ordered (ol) -->
    <h3>My Favorite Programming Languages:</h3>
    <ul>
        <li>Python</li>
        <li>JavaScript</li>
        <li>Java</li>
    </ul>
    
    <h3>Steps to Build a Website:</h3>
    <ol>
        <li>Plan the layout</li>
        <li>Write HTML structure</li>
        <li>Style with CSS</li>
        <li>Add interactivity with JavaScript</li>
    </ol>
    
    <!-- Forms: For user input -->
    <h3>Contact Us:</h3>
    <form action="#" method="POST">
        <label for="name">Name:</label><br>
        <input type="text" id="name" name="name" placeholder="Your Name"><br>
        <label for="email">Email:</label><br>
        <input type="email" id="email" name="email" placeholder="Your Email"><br>
        <label for="message">Message:</label><br>
        <textarea id="message" name="message" rows="4" placeholder="Your Message"></textarea><br>
        <input type="submit" value="Send Message">
    </form>
    
    <!-- Semantic HTML5: For better structure and SEO -->
    <article>
        <h3>My Latest Blog Post</h3>
        <p>Content of the blog post goes here...</p>
    </article>
    
    <footer>
        <p>&copy; 2025 My Awesome Page</p>
    </footer>
</body>
</html>

Lesson 2: CSS - Styling Your Webpage

CSS (Cascading Style Sheets) is used to control the presentation of your HTML documents. It dictates colors, fonts, spacing, layout, and much more.

Selectors, Box Model & Basic Layout (Flexbox):

  • CSS Selectors: Element, class, ID, attribute selectors
  • Box Model: Content, padding, border, margin
  • Units: px, %, em, rem, vw, vh
  • Flexbox: display: flex, flex-direction, justify-content, align-items
  • Grid: display: grid, grid-template-columns/rows
  • Responsive Design: Media queries for different screen sizes
Previous Lesson Next Lesson: JavaScript 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>CSS Styling</title>
    <style>
        body {
            font-family: 'Inter', sans-serif;
            background-color: #2c3e50;
            color: #ecf0f1;
            margin: 20px;
            padding: 20px;
        }
        /* Element Selector */
        h1 {
            color: #3498db;
            text-align: center;
            margin-bottom: 20px;
        }
        /* Class Selector */
        .card {
            background-color: #34495e;
            border: 2px solid #2980b9;
            padding: 20px;
            margin: 10px;
            border-radius: 8px;
            box-shadow: 0 4px 8px rgba(0, 0, 0, 0.2);
        }
        /* ID Selector */
        #main-container {
            display: flex; /* Makes children flex items */
            flex-wrap: wrap; /* Allows items to wrap to next line */
            justify-content: center; /* Centers items horizontally */
            align-items: flex-start; /* Aligns items to the start vertically */
            gap: 20px; /* Space between flex items */
            padding: 20px;
            background-color: #1a202c;
            border-radius: 10px;
        }
        /* Hover Effect (Pseudo-class) */
        .card:hover {
            transform: translateY(-5px); /* Move up slightly */
            box-shadow: 0 8px 16px rgba(0, 0, 0, 0.4);
            transition: all 0.3s ease; /* Smooth transition */
        }
    </style>
</head>
<body>
    <h1>Styling with CSS: Box Model & Flexbox</h1>
    <div id="main-container">
        <div class="card">
            <h3>Card One</h3>
            <p>This is the content for card one.</p>
        </div>
        <div class="card">
            <h3>Card Two</h3>
            <p>This is the content for card two.</p>
        </div>
        <div class="card">
            <h3>Card Three</h3>
            <p>This is the content for card three.</p>
        </div>
    </div>
</body>
</html>

Lesson 3: JavaScript - Adding Interactivity

JavaScript is the programming language that brings your webpages to life! It allows you to create dynamic content, handle user interactions, and much more.

Variables, Data Types, Operators, Control Flow & Functions:

  • Variables: let, const, var (modern: let, const)
  • Data Types: String, Number, Boolean, Array, Object, null, undefined
  • Operators: Arithmetic (+, -, *, /), Comparison (==, ===, !=, !==), Logical (&&, ||, !)
  • Control Flow: if/else, for/while loops, switch statements
  • Functions: Reusable blocks of code with parameters and return values
  • DOM Manipulation: Access and modify HTML elements with JavaScript
Previous Lesson Next Lesson: Responsive Design 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>JavaScript Basics</title>
<style>
    body { font-family: 'Inter', sans-serif; background-color: #2c3e50; color: #ecf0f1; padding: 20px; }
    .card { background-color: #34495e; border: 2px solid #2980b9; padding: 20px; margin: 10px; border-radius: 8px; box-shadow: 0 4px 8px rgba(0, 0, 0, 0.2); }
    button { background-color: #3498db; color: white; padding: 10px 20px; border: none; border-radius: 4px; cursor: pointer; transition: background-color 0.3s ease; }
    button:hover { background-color: #2980b9; }
    #jsOutput { margin-top: 15px; padding: 10px; background-color: #1a202c; border-radius: 4px; }
</style>
</head>
<body>
    <h1>JavaScript Fun!</h1>
    <div id="jsOutput" class="text-green-300">Watch the magic happen...</div>
    <div class="card">
        <h3>JavaScript Basics</h3>
        <p>JavaScript is a versatile programming language that adds interactivity to web pages.</p>
    </div>
    <button id="myJsButton">Click Me!</button>
    
    <script>
        // 1. Variables: Store data
        let userName = "Harry"; // Modern way, block-scoped
        const PI = 3.14; // Constant, cannot be reassigned
        var oldVar = "Avoid this in new code"; // Old way, function-scoped
        
        // 2. Data Types:
        let str = "Hello"; // String
        let num = 123;     // Number
        let bool = true;   // Boolean
        let arr = [1, 2, 3]; // Array
        let obj = { key: "value" }; // Object
        let nothing = null; // Null
        let undefinedVar; // Undefined
        
        // 3. Operators: Perform operations
        let sum = 10 + 5; // Addition
        let product = 10 * 5; // Multiplication
        let isEqual = (10 === "10"); // Strict equality (false)
        let isGreater = (10 > 5 && 8 < 10); // Logical AND (true)
        
        // 4. Control Flow: if/else, loops
        let age = 20;
        if (age >= 18) {
            console.log("You are an adult.");
        } else {
            console.log("You are a minor.");
        }
        
        // Loop: Iterate
        console.log("Counting to 3:");
        for (let i = 0; i < 3; i++) {
            console.log(i + 1);
        }
        
        // 5. Functions: Reusable blocks of code
        function greet(name) {
            return "Hello, " + name + "!";
        }
        console.log(greet(userName)); // Output: Hello, Harry!
        
        // DOM Manipulation: Interact with HTML elements
        const jsButton = document.getElementById('myJsButton');
        const jsOutput = document.getElementById('jsOutput');
        let clickCount = 0;
        
        jsButton.addEventListener('click', function() {
            clickCount++;
            jsOutput.innerHTML = "Button clicked " + clickCount + " times!";
            jsOutput.style.color = '#4ade80'; // Tailwind green-400 equivalent
        });
    </script>
</body>
</html>

Lesson 4: Responsive Design - Adapting to All Devices

Responsive web design ensures your website looks and functions great on any device, from tiny smartphones to large desktop monitors.

Key Concepts & Media Queries:

  • Viewport Meta Tag: Always include <meta name="viewport" content="width=device-width, initial-scale=1.0"> in your HTML <head>. This is crucial.
  • Fluid Layouts: Use relative units like percentages (%), vw (viewport width), em, rem for widths, heights, and font sizes. Avoid fixed px values.
  • Flexible Images: max-width: 100%; height: auto; makes images scale down without distortion.
  • Media Queries: These are CSS rules that apply styles based on device characteristics (like screen width).
Previous Lesson Next Lesson: Mini Project 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Responsive Example</title>
    <style>
        body { font-family: sans-serif; text-align: center; padding: 20px; }
        .container {
            background-color: #3498db;
            color: white;
            padding: 20px;
            margin: 20px auto;
            max-width: 800px;
            border-radius: 8px;
            box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);
        }
        /* Default styles for small screens (mobile-first) */
        .container h2 {
            font-size: 1.5rem; /* 24px */
        }
        .container p {
            font-size: 1rem; /* 16px */
        }
        /* Media query for tablets and larger (min-width: 768px) */
        @media (min-width: 768px) {
            .container {
                background-color: #2ecc71; /* Green on tablet */
                max-width: 600px;
            }
            .container h2 {
                font-size: 2rem; /* 32px */
            }
            .container p {
                font-size: 1.125rem; /* 18px */
            }
        }
        /* Media query for desktops and larger (min-width: 1024px) */
        @media (min-width: 1024px) {
            .container {
                background-color: #e74c3c; /* Red on desktop */
                max-width: 900px;
            }
            .container h2 {
                font-size: 2.5rem; /* 40px */
            }
            .container p {
                font-size: 1.25rem; /* 20px */
            }
        }
    </style>
</head>
<body>
    <div class="container">
        <h2>Resize your browser to see changes!</h2>
        <p>This box changes color and size based on screen width.</p>
    </div>
</body>
</html>
